使用hexo与github搭建白嫖个人博客

这是我的博客地址: https://liuyan011006.github.io/

注意:这个白嫖的重点是你可以访问github,如果访问不了的话,个人博客就白嫖不成功了!!!(Hexo可以和github搭配,可以省去租赁服务器的麻烦)

一,搭建准备

  • 需要注册一个github账号
  • 需要安装Git
  • 需要安装NodeJs

二,创建一个存放博客仓库

创建

  • 新建一个名字为你的用户名.github.io的仓库

    比如你的用户名为 xxx 仓库就为 xxx.github.io

    注意:名字必须为用户名.github.io 否则不成功

配置ssh密钥

  • 为什么要配置这个呢？因为你提交代码肯定要拥有你的github权限才可以，但是直接使用用户名和密码太不安全了，所以我们使用ssh key来解决本地和服务器的连接问题。

    简单的来说就是给予你当前电脑访问你GitHub 仓库的权限 ~

    1. 在 git bash here 中创建一个ssh密钥

      指令：cd ~/.ssh

      如果本地有的话，它会自动进入到了ssh目录下；

      若没有则创建： ssh-keygen -t rsa -C “邮箱” 此处邮箱填写自己的邮箱

      然后enter三连

      生成两个文件： id_rsa id_rsa.pub 存放在 C:\Users\admin.ssh 文件夹下

      注意：pub文件放置的，就是我们的公开的ssh码，也是给管理员这个ssh，让他来添加

    2. 在Github上添加ssh

      Title:随意写

      Key:把生成的id_rsa.pub文件用记事本打开，复制全部内容粘贴到GitHub 仓库的SSH Keys 位置

至此，搭建准备已经做好，接下来让我们快速开始搭建博客吧 ~

三,本地安装Hexo博客框架,并推送到Github

1.1博客初始化

  1. 安装博客框架

  1. 在D盘或者其他盘创建HexoData(其他名字也可以),但是不要随意存放,这是你博客存放代码的地方

  2. 进入这个文件夹,右键打开Git Bash Here

    然后执行一下命令

    hexo init 初始化,生成一些文件node_modules等

    hexo g 生成命令

    hexo s 启动命令

    说明：hexo s 是开启本地预览服务，打开浏览器访问 http://localhost:4000 即可看到内容，说明博客初始化以及成功！

四,美化博客

1.1切换主题

  • 可以进入theme,挑选一个你中意的页面,然后将此文件下载到文件夹下的themes里

    git clone 地址链接

  • 主题下载完成后，将_config.yml站点配置文件中的 theme 值修改为你下载主题的文件名，此处为 hexo-theme-matery，那么值就修改为 theme: hexo-theme-matery
  • 网址修改：url: 的值为你的网址名，如 http://xxxx.github.io，如果有域名，则修改为你的域名即可;
  • 站点配置文件有个 per_page属性，建议修改为 6 的倍数，这样网站在适应设备时，有较好的显示效果。
